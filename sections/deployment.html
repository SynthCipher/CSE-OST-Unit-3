3.4 Deploying Applications

Making your web applications available on the internet or local server.

Requirements for Deployment

To deploy a PHP and MySQL application, you generally need:

  • Domain Name: e.g., www.mywebsite.com
  • Web Hosting: Server supporting Apache, PHP, and MySQL.
  • Access: FTP or a Control Panel (cPanel).

Local Server for Testing

Before going online, test locally using stacks like XAMPP, WAMP, or LAMP. They provide the necessary environment (Apache, PHP, MySQL).

Deploying Using XAMPP

  1. Install XAMPP and start Apache/MySQL.
  2. Place files inside the htdocs folder.
  3. Access via http://localhost/projectname.

Steps to Deploy Online

1

Create & Test

Develop your files and test thoroughly on your local server.

2

Upload Files

Upload your PHP files to the hosting server via FTP or File Manager.

3

Database Migration

Import your MySQL database (.sql file) using tools like phpMyAdmin on the server.

4

Final Connection

Update your PHP connection strings with the production database credentials.

Common Problems during Deployment

  • Database connection errors
  • Missing files or wrong paths
  • Incorrect database credentials
  • PHP version mismatch
