3.3 PHP & MySQLi

Mastering server-side scripting and relational database management.

Introduction to PHP

PHP (Hypertext Preprocessor) is a server-side scripting language. It runs on the server and is used to process forms, connect to databases, and generate dynamic content.

Why PHP?

  • Open source and free
  • Works well with MySQL
  • Used by giants like WordPress and Wikipedia
  • PHP code is not visible to the user (Server-side)

Embedding PHP in HTML

<!DOCTYPE html> <html> <body> <h1> <?php echo "Welcome to PHP"; ?> </h1> </body> </html>

PHP Variables

<?php $name = "Rahul"; $age = 20; echo "Name: " . $name; echo "<br>Age: " . $age; ?>

Form Handling

PHP can collect data from HTML forms using $_POST or $_GET.

// HTML Form <form method="post" action="welcome.php"> Name: <input type="text" name="username"> <input type="submit"> </form> // welcome.php <?php $name = $_POST['username']; echo "Welcome " . $name; ?>

MySQLi (MySQL Improved)

MySQL is an open-source relational database. MySQLi is the extension used to connect PHP with MySQL.

Connecting to Database

<?php $servername = "localhost"; $username = "root"; $password = ""; $database = "college"; $conn = new mysqli($servername, $username, $password, $database); if ($conn->connect_error) { die("Connection failed: " . $conn->connect_error); } echo "Connected successfully"; ?>

CRUD Operations

  • INSERT: INSERT INTO students (name, course) VALUES ('Aman', 'BCA')
  • SELECT: SELECT * FROM students
  • UPDATE: UPDATE students SET course='MCA' WHERE id=1
  • DELETE: DELETE FROM students WHERE id=1

Steps to use MySQLi

  1. Create a database
  2. Create a table
  3. Connect PHP with MySQL
  4. Execute SQL queries
  5. Display results
