3.2 Building Dynamic Webpages

Using HTML, CSS, and JavaScript to build the foundation of the web.

HTML (HyperText Markup Language)

HTML is used to create the structure of a webpage. It defines elements like headings, paragraphs, images, tables, and forms.

<!DOCTYPE html> <html> <head> <title>My Webpage</title> </head> <body> <h1>Welcome</h1> <p>This is a simple webpage.</p> </body> </html>

CSS (Cascading Style Sheets)

CSS is used to design and style webpages. It controls colors, fonts, borders, layout, and spacing.

<style> body { background-color: lightblue; font-family: Arial; } h1 { color: darkblue; } </style>

JavaScript

JavaScript is used to make webpages interactive. It can show alerts, validate forms, and respond to button clicks.

<script> function greet() { alert("Welcome to the website!"); } </script> <button onclick="greet()">Click Me</button>

Simple Dynamic Webpage Example

Combining all three to change content on the fly.

<h1>Student Information</h1> <p id="msg">Welcome Student</p> <button onclick="changeText()">Change Text</button> <script> function changeText() { document.getElementById("msg").innerHTML = "You clicked the button"; } </script>
