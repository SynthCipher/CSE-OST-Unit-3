3.1 Introduction to Web Development

Mastering the fundamentals of open-source technologies and architecture.

What is Web Development?

Web development is the process of creating websites and web applications that run in a browser.

Open-Source Technologies

Open-source technologies are software tools whose source code is freely available. Developers can use, modify, and distribute them without paying license fees.

Examples of Open-Source Web Technologies

  • HTML / CSS / JavaScript: The core of frontend.
  • PHP: Server-side scripting.
  • MySQL: Database management.
  • Apache Web Server: Serving the content.
  • XAMPP / WAMP: Local development stacks.

Why use Open-Source?

  • Free to use (No license fees)
  • Easy to learn
  • Supported by large communities
  • Suitable for both small and large websites

Components of a Web Application

1

Front-End

The visible part seen by the user. Built using HTML, CSS, and JavaScript.

2

Back-End

Handles server-side processing. Built using PHP.

3

Database

Stores information such as usernames, passwords, and product details. MySQL is commonly used.

Client-Server Architecture

In web development:

  • The Client: The web browser (Chrome, Firefox, etc.)
  • The Server: Stores the website and processes requests.

The Request-Response Flow

  1. User requests a webpage.
  2. Server processes the request.
  3. Server sends the webpage back to the browser.

Example Scenario:

A user fills a login form. PHP checks the username and password from the MySQL database and sends the result back to the client.
