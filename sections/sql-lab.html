SQL & Database Lab

Hands-on scripts to create databases, tables, and perform CRUD operations.

1. SQL Setup Script (.sql)

Use in phpMyAdmin SQL tab
-- Create and Use Database CREATE DATABASE IF NOT EXISTS college; USE college; -- Create Table CREATE TABLE IF NOT EXISTS students ( id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(50) NOT NULL, course VARCHAR(50) NOT NULL ); -- Insert Initial Data INSERT INTO students (name, course) VALUES ('Rahul Sharma', 'BCA'), ('Ali Khan', 'MCA');

2. PHP Database Connection & Display

Save as db_demo.php
<?php $conn = new mysqli("localhost", "root", "", "college"); if ($conn->connect_error) { die("Connection failed: " . $conn->connect_error); } $sql = "SELECT * FROM students"; $result = $conn->query($sql); if ($result->num_rows > 0) { while($row = $result->fetch_assoc()) { echo "ID: " . $row["id"]. " - Name: " . $row["name"]. "<br>"; } } else { echo "0 results"; } $conn->close(); ?>

How to run these?

  1. Start Apache and MySQL in XAMPP.
  2. Open phpMyAdmin and paste the SQL script into the SQL tab.
  3. Save the PHP code as a file in your htdocs folder.
  4. Run it by visiting localhost/yourfilename.php.
